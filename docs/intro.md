# Welcome to ARGUS

<div style={{display: 'flex', justifyContent: 'center', alignItems: 'center', margin: '2rem 0', borderRadius: '12px', overflow: 'hidden'}}>

ARGUS is a next-generation self-custody wallet that replaces single-key trust with **environment-based security**.

Instead of trusting a private key alone, ARGUS verifies *real-world context* before allowing any sensitive transaction to execute.
